DecorPalette user guide

This guide, written by DecorPalette's creator Öner Koray, walks through every feature so you can move from a single anchor color to a finished, accessible scheme.

1. Choosing a base color

Use the color picker or type a six-digit hex value into the field beside it. The two inputs stay in sync, and an invalid hex is rejected with a validation message rather than breaking the palette.

2. Selecting a harmony

Each harmony applies a different set of hue rotations to your base color. Analogous stays close for calm, tonal rooms; complementary and triadic create bolder contrast for accent-driven spaces; monochromatic varies only lightness for a refined single-hue look.

3. Locking and shuffling

Press the lock button on any swatch to keep it fixed. Pressing Space (or the Shuffle button) regenerates only the unlocked colors, letting you converge on a scheme one decision at a time.

4. Checking contrast

The contrast checker computes the WCAG relative-luminance ratio between a text color and a background color. Aim for 4.5:1 for body text and 3:1 for large text to meet WCAG 2.2 AA.

5. Exporting and saving

Copy the whole palette as CSS custom properties or a JSON array, or save it to your local library. Saved palettes persist in your browser and can be reloaded into the generator at any time.
